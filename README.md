# MAKO

**A simple, sharp programming language made to be easy to learn and powerful enough to build real software.**

MAKO is an upcoming programming language focused on keeping code easy to read, easy to type, and easy to remember while still aiming to be useful for real projects.

The goal is to mix the simplicity of languages like Python and Lua/Luau with a more structured C#/C-style feel. MAKO is planned to be used for things like tools, backend software, rendering experiments, system utilities, and eventually larger native projects.

> [!WARNING]
> MAKO is currently in early planning and is not released yet.
> The syntax, compiler, tooling, and standard library are still being designed.

## What MAKO Wants To Be

MAKO is being designed around a few simple ideas:

* Easy to learn
* Easy to remember
* Easy to type on a normal keyboard
* Simple enough for beginners
* Powerful enough to grow into real software
* Clean and structured without too much boilerplate

## Example

```mako
script "Hello";

main() {
    print "Hello from MAKO";
}
```

## File Extension

MAKO source files use:

```txt
.mko
```

Example:

```txt
main.mko
```
